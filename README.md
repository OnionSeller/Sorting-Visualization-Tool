
# Sorting Visualization Tool in React.js

*For online demonstration* -> Todo

This project is a web application built with React.js (Vite) that allows users to see different sorting algorithms in action, idealy used for educational purposes.

## Features

- Different Sorting Algorithms: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, Radix Sort, Bucket Sort
- Visualization of the sorting process in real-time by utilizing 2D rendered bars 
- Ability to adjust the speed of the visualization
- Detailed explanations for each algorithm, including a brief description, key steps, and efficiency
- A table comparing each algorithm's performance in terms of the time complexity from best to worst and the space complexity


## Installation

- This project requires Node.js (version 12 or higher). If not already installed, you can install it here: https://nodejs.org/en
First clone this repository to your local machine:
```bash
  git clone Todo
```
Then install the dependencies:
```bash
npm install
```
    
## Usage

To start, run:
```javascript
npm run dev
```
